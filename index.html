How it works

Four steps.
Zero effort.

Install once. Every merged PR automatically keeps your docs in sync.

01 — DETECT

PR merged

GitHub sends a webhook the instant a PR is merged. We receive it in under 100ms.

02 — ANALYSE

Diff extracted

We scan the changed files and extract every modified function, class, and type signature.

03 — MATCH

Docs mapped

We find exactly which doc sections mention the changed symbols — no false positives.

04 — REWRITE

PR opened

AI rewrites only the affected sections in your team's voice. A PR opens for human review.
